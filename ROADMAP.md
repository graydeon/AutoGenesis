# Roadmap

## v0.1.0 (Current)

All 8 packages: core runtime, CLI, tools, MCP, tokens, optimizer, security, plugins. The complete framework foundation.

- Single-threaded async agent loop (Claude Code pattern)
- 3-tier model routing via LiteLLM (fast/standard/premium)
- 12 built-in tools with progressive disclosure
- MCP client and server from day one
- Token counting, budgeting, and exact-match caching
- Self-improving prompt engine with constitutional safety
- Input/output guardrails, audit logging, security scanning
- Plugin architecture with entry point discovery
- Full CLI with Typer + Rich
